# Place and Plan

Eine AR-App, die Menschen bei der Einrichtung und Planung von Bildern helfen soll.

## Setup
> [!IMPORTANT]
> Bitte das Projekt NICHT über "Download ZIP" herunterladen, da dann nicht alle Dateien heruntergeladen werden.

1. Quellcode über git oder [Releases](https://github.com/Lilli-km/place-and-plan/releases/) herunterladen.
2. In Unity Hub importieren und öffnen.
3. Die Scene "SampleScene" im Ordner Scenes/SampleScene öffnen.
4. Unter File > Build Profiles die Platform Android auswählen und auf "Switch Platform" klicken.
5. Projekt mit "Play" starten und evtl. die Auflösung anpassen.
